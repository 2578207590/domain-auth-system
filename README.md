# 🛡️ 域名/IP 授权系统

> 基于 **PHP + MySQL** 的轻量级域名与 IP 授权管理系统。支持卡密生成与激活、域名/IP 续费、泛域名授权、客户端自动验证。

---

## 📋 功能总览

| 模块 | 功能 |
|------|------|
| 🎫 卡密管理 | 生成、导入、导出卡密；支持批量删除、批量修改有效期 |
| 🌐 域名/IP 管理 | 添加、修改、删除域名或 IP；支持续费、封禁/解封、泛域名授权 |
| 🔐 客户验证 | 网页前端 JS 自动检测域名或 IP 授权状态，未授权/已到期弹窗激活 |
| ⚙️ 系统设置 | 修改管理员账号密码（bcrypt 加密存储） |
| 📋 操作日志 | 记录所有操作，支持搜索、清理 |
| 📊 统计面板 | 授权数量、到期数量、封禁数量、卡密使用率 |

## 🎫 卡密管理

### 生成卡密

支持设置**自定义授权天数**：

- 系统默认（365 天）
- 30 / 90 / 180 / 365 天预设
- **永久有效**
- **自定义天数**（任意数字，如 720 天）

### 批量操作

- 批量删除已选卡密
- 批量修改有效期（支持自定义天数）
- 清空所有已使用卡密

### 导出卡密

- 纯码导出
- 完整导出（含域名、使用时间）
- 按状态筛选导出（未使用 / 已使用）

## 🌐 域名 & IP 管理

### 授权方式

- **域名授权** — 标准域名（如 `example.com`）
- **IP 授权** — 公网或内网 IP 地址（IPv4 / IPv6）
- **泛域名** — 通配符匹配子域名（`*.example.com`），需管理员手动添加
- 到期前 30 天自动显示 **"即将到期"** 橙色提示

### 续费

- 后台一键续费：+30 / +90 / +365 天或设为永久
- 续费时**叠加到期时间**
- 已到期授权续费后恢复正常状态

### 严格格式校验

| 输入 | 结果 | 说明 |
|------|------|------|
| `abc.com` | ✅ | 标准域名 |
| `192.168.1.1` | ✅ | IPv4 地址 |
| `2001:db8::1` | ✅ | IPv6 地址 |
| `*.abc.com` | ✅ | 泛域名 |
| `localhost` | ❌ | 本地回环拒绝 |
| `随便输入` | ❌ | 任意字符串拒绝 |

## 🔐 客户端验证

部署 `auth.js` / `auth_en.js` 到需要授权保护的网页中：

- 页面加载时自动检测当前域名/IP 授权状态
- **未授权** → 弹出遮罩层，显示激活界面
- **已到期** → 弹出遮罩层，显示续费界面
- **已封禁** → 弹出遮罩层，禁止访问
- **授权正常** → 无动作，页面正常显示

### 激活流程

1. 用户输入卡密
2. 点击激活按钮（含 loading 动画）
3. 成功后自动刷新页面
4. 显示真实到期时间（不硬编码 365 天）

## ⚙️ 系统管理

### 安全特性

- **CSRF 保护** — 所有 POST 操作使用随机 Token 验证
- **会话超时** — 后台 30 分钟无操作自动退出
- **登录限制** — 5 次错误密码后冻结 15 分钟
- **密码加密** — bcrypt 加密存储
- **XSS 防护** — 所有输出 htmlspecialchars 转义
- **事务保护** — 卡密激活使用数据库事务
- **输入校验** — 仅接受标准域名或合法 IP
- **操作日志去重** — 同操作每日只记一条

## 🚀 安装部署

### 环境要求

- **PHP** >= 7.0（推荐 7.4+）
- **MySQL** 5.6+ / MariaDB
- 需 `mysqli` 扩展和 `password_bcrypt`

### 安装步骤

访问 `http://your-domain.com/install.php`

| 字段 | 说明 |
|------|------|
| 数据库地址 | 通常为 localhost |
| 数据库用户名 | MySQL 用户 |
| 数据库密码 | MySQL 密码 |
| 数据库名 | 需提前创建 |
| 管理员账号 | 后台登录用 |
| 管理员密码 | 后台登录用 |

安装后自动跳转到 `admin.php` 后台。

### 配置客户端验证

修改 `auth.js` / `auth_en.js` 中的 API 地址：

```javascript
const API = "https://your-domain.com/api.php";
```

## 🔌 API 接口

### 检查授权状态

```
GET api.php?act=check&domain=xxx
```

```json
{"code":1,"expire_time":"2027-05-06"}   // 已授权
{"code":2,"msg":"已到期"}               // 已到期
{"code":-1,"msg":"已封禁"}              // 已封禁
{"code":0}                              // 未授权
```

### 卡密激活

```
GET api.php?act=active&domain=xxx&code=CARD123
```

```json
{"code":1,"msg":"激活成功","expire_time":"2027-05-06"}
```

### 授权列表

```
GET api.php?act=list
```
